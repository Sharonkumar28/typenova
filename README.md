## 📖 Overview

**TypeNova** is a modern typing application designed to improve typing speed and accuracy while making practice enjoyable.

The platform offers two engaging modes:

- ⌨️ **Classic Typing Test** – Practice typing and monitor your performance.
- 🎮 **Arcade Typing Game** – Destroy enemies by typing words correctly before they reach you.

Whether you're a beginner or an experienced typist, TypeNova provides an interactive and fun learning experience.

## ✨ Features

- ⚡ Real-time WPM Tracking

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Sharonkumar28/TypeNova.git
```

Go to the project directory

```bash
cd TypeNova
```

Open **index.html** in your browser.

No installation or additional dependencies are required.
