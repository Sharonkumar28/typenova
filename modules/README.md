# 🪐 TypeNova

TypeNova is a futuristic hybrid typing platform that combines a classic typing test experience with an arcade-style typing game.

## 🚀 Features

- Real-time WPM tracking
- Accuracy calculation
- Arcade shooter typing mode
- Combo and scoring system
- Theme switching
- Audio feedback system
- Virtual keyboard visualizer
- Local leaderboard storage

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Canvas API
- Web Audio API
- LocalStorage
